Auto-Assignment Mode: Smart & Simple

Perfect for Beginners

Auto-Assignment uses AI to place tables optimally based on party preferences and room constraints.

What it does:

  • Calculates optimal table sizes
  • Places tables in preferred locations
  • Handles collision detection
  • Groups parties when requested

Your preferences matter:

  • Round vs rectangular tables
  • Near windows, bar, or stage
  • Quiet areas or center room
  • Group seating options

Manual Mode: Full Creative Control

For Experienced Planners

Take complete control over your event layout with our intuitive drag-and-drop interface.

Create Tables:

  • Drag from table palette
  • Drop anywhere on layout
  • Choose sizes: 4-10 guests
  • Round or rectangular

Table Controls:

  • Hover to see controls
  • Rotate in 45 increments
  • Resize capacity (4-12)
  • Delete when needed

Assign Parties:

  • Select parties first
  • Click target table
  • System warns of overcrowding
  • View assignments anytime

Professional Reports & Best Practices

Hostess Reports

Generate comprehensive reports for your hostess team with all the details they need.

Reports Include:

  • Total guests, adults, children
  • Members vs non-members
  • Table-by-table assignments
  • Assignment method used
  • Unassigned parties (if any)

Best Practices

  • Plan 15% extra capacity for walk-ins
  • Use Auto-Assignment first, then fine-tune manually
  • Double-click tables to see who's assigned
  • Generate reports before the event
  • Keep the CAD layout for reference

Troubleshooting & Quick Reference

Common Issues

Tables won't move:

Switch to Manual mode to enable dragging

Can't assign parties:

Select parties first, then click table

Tables overlapping:

Drag to reposition or use rotation

Missing CAD background:

Ensure cad_layout.png is uploaded

Quick Reference

Auto vs Manual: Toggle in mode selector
Rotate table: Hover rotation button
Resize table: Hover resize button
Delete table: Hover delete button
Table info: Double-click table
Clear all: Reports & Actions panel
Generate report: Hostess Report button
